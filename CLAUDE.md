# Daily Trading Research Agent

This repo contains skills and scripts for automated daily growth stock research.

## Environment Variables Required
- `FMP_API_KEY`: Financial Modeling Prep API key (for CANSLIM + VCP screeners)

## Config
Edit `config.json` to set account size and email.

## Skills Location
All skills are in `.claude/skills/`. The agent uses them automatically.

## Report Language
Write all reports in Chinese (中文). Use English only for stock ticker symbols and technical terms.
